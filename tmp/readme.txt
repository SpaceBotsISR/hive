### Description of the contents in this folder

base<number>.bag - each of these files represents the processed result from data<number>.bag, obtained in the granite lab at NASA Ames Research Center's Granite lab.

svr_model<number>.bin - corresponds to a trained svr model. The number identifies the corresponding SVR script.

neural_model<number>.bin - the same as the previous one, but using neural networks.

calibration.bin - calibration file for the granite lab with correction
basecalibration.bin - calibration file for the granite lab without correction

hive_environment.json - parameters required to run the system
